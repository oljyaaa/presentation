3-рівнева архітектура

Єдине серверне ядро — абсолютна синхронізація між усіма точками входу

🖥️ Frontend Layer
📱
React Native (Expo)
Мобільний додаток для iOS/Android. Ролі: creator, admin, master. TypeScript.
💻
Vanilla JS + HTML/CSS
Адмін-панель у браузері. Дві версії: admin.html та creator.html. Phosphor Icons, Nunito.
🌐
React + TypeScript
Клієнтський сайт (Lovable AI). Vite, Tailwind, SPA-маршрутизація.
⟷
⚙️ API Layer
🔧
PHP REST API
Єдина точка входу — index.php. Actions: get_all_data, add_record, update_record, check_new_appointments, upload_avatar...
🔐
Автентифікація
Сесійна авторизація. Ролі: creator (повний доступ), admin, worker. JWT-like перевірка через localStorage / AsyncStorage.
📤
FormData + JSON
Всі запити — FormData POST або GET з параметрами. Відповідь завжди JSON із полем status.
⟷
🗄️ Data Layer
🐬
MySQL
Централізована реляційна БД. Таблиці: users, appointments, archive, callbacks, services, categories, history.
📁
Файлове сховище
Аватари майстрів зберігаються у /uploads/ на сервері. Посилання в БД.
🌍
Apache + Хостинг
Реальний shared-хостинг. thebeauty-room.com. SSL/HTTPS, PHP 8+, MySQL 5.7+.

Особливості мобільного додатку

Фішки, що роблять роботу майстрів справді зручною

🔔
Polling-сповіщення
Кожні 15 секунд перевіряє нові callback-заявки з сайту. При новій заявці — миттєве локальне push-сповіщення з іменем та телефоном клієнта.
setInterval(check, 15000) → scheduleNotificationAsync()
📅
Інтерактивний календар
Місячний календар із кольоровими крапками по майстрах. Натискання на день — bottom-sheet із переліком записів, згрупованих по майстрах.
handleDayPress → DateBottomSheet (slide-up animation)
🖨️
PDF-звіти
Генерація HTML-шаблону з усіма записами → конвертація в PDF через expo-print → sharing або відкриття у файловому менеджері.
expo-print → printToFileAsync() → shareAsync()
🌙
Темна / Світла тема
Дві повноцінні теми (LIGHT/DARK) з усіма кольорами. Перемикач прямо у профілі через кнопку місяця/сонця в шапці.
const T = isDark ? DARK : LIGHT
📸
Аватар профілю
Вибір фото з галереї через expo-image-picker. Обрізка 1:1, стиснення до 85%. Завантаження на сервер через FormData multipart.
ImagePicker → FormData.append(avatar) → API
📞
Конвертація заявок
Callback-заявки з сайту можна одним кліком конвертувати в офіційний запис: дані клієнта підставляються автоматично у форму запису.
handleConvertCall → setPreFillClient → setFormVisible

Особливості адмін-панелі

Ергономіка і потужність в одному інтерфейсі

🔍
Глобальний пошук (Ctrl+K)
Пошук по всій базі: записи, архів, callback. Результати — у випадаючому списку з миттєвим переходом до запису. Клавіатурний шорткат як у Notion.
search.js — debounce input → query API → dropdown render
📆
Big Calendar (Creator)
Повноцінний місячний/тижневий календар з кольорами по майстрах. При кліку на день — деталі записів у боковій панелі.
bigCalGrid → loadCalendarData() → renderLegend()
📊
Аналітика та звіти
Графік записів за тиждень. Фільтр по місяцю, майстру, статусу. Підрахунок доходу, втраченого доходу. Друк звіту (CSS print rules).
reportMonthSelect → filter → reportIncome / reportLostIncome
👥
Управління користувачами
Список акаунтів із ролями та правами. Редагування ім'я, email, ролі, пароля. Тонке налаштування доступів по категоріях послуг.
formEditUser → permissions: ["all", "Масаж", "archive"...]
📞
"Взяти дані" з callback
Кнопка у формі нового запису автоматично підтягує дані із callback-заявки. Клієнт обслуговується без ручного перенесення даних.
takePendingCallback() → fillForm(client, phone)
🎨
Теми оформлення
Light / Dark / System — три теми через CSS-змінні. Вибір зберігається в localStorage. Smooth-transition між темами без перезавантаження.
setTheme(t) → document.documentElement.setAttribute("data-theme", t)

Процес розробки

Ітеративний підхід від аналізу вимог до введення в експлуатацію

Жовтень 2024
📋 Аналіз вимог та планування
Інтерв'ю з власницею салону. Виявлення больових точок. Проєктування БД (8 таблиць). Вибір технологічного стеку. Визначення ролей користувачів.
Листопад 2024
⚙️ Розробка PHP API та бази даних
Налаштування хостингу, домену. Написання REST API (20+ actions). CRUD для записів, архіву, callback-заявок, прайсу, користувачів, журналу.
Грудень 2024
💻 Розробка адмін-панелі (Vanilla JS)
Двофайлова панель (admin + creator). Глобальний пошук, теми, календар, аналітика. Усі JS-модулі вручну без фреймворків.
Січень 2025
📱 Розробка мобільного додатку (React Native)
Рольова архітектура, календар, polling-сповіщення, PDF-звіти, аватари. TypeScript + Expo. Unit-тести Jest. Збірка APK через EAS.
Лютий 2025
🌐 Клієнтський сайт (Lovable AI) + QA
Дослідження Lovable як інструменту. Редизайн сайту на React+TS. White/Black Box тестування. Виправлення багів (BR-004 та ін.). Введення в експлуатацію.
